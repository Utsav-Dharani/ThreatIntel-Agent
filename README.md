# ThreatIntel Agent

ThreatIntel Agent is an AI-powered cyber threat intelligence workspace that transforms threat reports into structured intelligence briefs. It supports text input, public URL analysis, and PDF upload, then generates risk scoring, MITRE ATT&CK mapping, indicators of compromise, evidence-backed findings, recommendations, and analyst-ready reports.

> This is a research and portfolio project built for defensive security analysis. AI-generated findings may be incomplete or inaccurate and should be reviewed before real-world security use.

## Project Overview

ThreatIntel Agent is designed to help security analysts, students, and recruiters understand how AI agents can support cyber threat intelligence workflows.

The system accepts a threat report and processes it through an AI-assisted analysis pipeline:

```txt
Input Source
→ Content Extraction
→ Threat Intelligence Agent
→ Entity / IOC / CVE Extraction
→ MITRE ATT&CK Mapping
→ Risk Scoring
→ Recommendation Generation
→ Evidence-backed Report
→ Oracle Database Storage
→ React Dashboard Display
